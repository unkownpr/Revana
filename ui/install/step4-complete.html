What to do next

1. Log in — Use your admin account at the login page.

2. Admin Panel — Go to /admin to manage your game. Configure seasons, edit the map, manage users, set game rules.

3. Create a Season — Admin → Seasons → Create your first game season before players can register and play.

4. Design the Map — Admin → Map Editor to paint your world with plains, water, and resource tiles.

5. Open Registration — Admin → Config → Enable "Allow Registration" when you are ready to invite players.

6. Share your server URL — Let the conquest begin!

⚠ Security Checklist

• Keep config.ini secure — it contains your database credentials.

• Make sure config.ini is in .gitignore before pushing to any repository.

• Delete install.lock only if you need to reinstall — it re-enables the installer.

• Use a strong admin password and change it if others helped with setup.
