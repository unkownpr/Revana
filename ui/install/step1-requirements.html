Troubleshooting Tips

• PHP 8.0+ — Update your PHP version via your control panel or hosting provider.

• PDO / PDO MySQL — Enable extension=pdo_mysql in your php.ini.

• GD Library — Enable extension=gd in your php.ini.

• config.ini writable — Run chmod 666 config.ini or create an empty file with write permission.
