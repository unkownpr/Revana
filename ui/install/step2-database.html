Installation Wizard — Step 1 of 3: Database Connection Error

⚠ Connection Failed

{{ @error }}

Common causes & fixes:

• Wrong credentials — double-check host, port, username and password.

• Database does not exist — create it first:
  CREATE DATABASE revana CHARACTER SET utf8mb4;

• User lacks privileges — grant them:
  GRANT ALL ON revana.* TO 'user'@'localhost';

• MySQL/MariaDB service is not running — start it with service mysql start.


 Please correct your database details below and try again.
