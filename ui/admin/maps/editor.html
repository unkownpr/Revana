ⓘ How to use the Map Editor
Painting Tiles

Select a terrain brush from the Palette below. Then click any tile on the map to paint it with that terrain. The currently selected brush is shown in the toolbar.

Navigation

Use WASD or ←↑→↓ to move the view. Or type coordinates into the Jump to fields and click Go. You can also click the direction arrows on the map image edges.

Tile Types

Type 1 = Land (plains, resource fields — walkable, farmable)
Type 2 = Water (rivers, lakes — impassable, decorative)
Type 3 = Town slot (where players can found towns)
The subtype distinguishes different visuals within the same type.

Generate Random Destructive

Fills the entire map with randomly generated terrain. Existing tiles are overwritten. Use this to create a starter map quickly, then paint over specific areas.

Custom Brushes

Click "+" in the palette to upload your own terrain image. It will automatically be normalized to an 80×80 isometric tile. Custom brushes can be deleted (except built-in ones).

Biome Rules

The Biome Rules section controls how towns look in-game based on their surrounding terrain. Set which background image and map icon to use per terrain type or faction.
